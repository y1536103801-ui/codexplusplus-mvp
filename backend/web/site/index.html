FAQ

常见问题

这是 OpenAI 官方产品吗？

不是。Codex+++ 是独立开发的自动化客户端和服务，帮助普通用户更简单地使用 Codex；官方 Codex 桌面应用仍由 OpenAI 提供。

为什么购买后还需要人工联系？

当前没有接入自动支付渠道。人工确认可以确保收款、账号创建和 Token 入账保持一致，避免出现没有实际付款却显示成功的情况。

一个账号用完后需要重新登录吗？

通常不需要。后端会在可用号池中动态选择路由；当前账号额度结束或不可用时，会尝试切换其他可用账号。

会覆盖我电脑上自己的 Codex 登录吗？

不会强制覆盖。检测到不属于本服务的个人登录时，客户端会提示先退出；只有没有登录，或账号属于本服务号池时才继续准备。

Windows 和 macOS 的账号通用吗？

通用。两种客户端连接同一后端，余额、用量、设备与号池路由规则保持一致；同一设备仍按后端并发规则管理。

如何继续充值 Token？

登录桌面客户端后，可以选择管理员启用的 Token 充值项并提交申请。管理员确认后，Token 会加入当前账号余额。
