# 新手教学 UI

## 目标

为第一次使用 Codex 的用户提供任务模板、提问方式、结果确认和安全提醒。

## 涉及项目

- CodexPlusPlus-main：`apps/codex-plus-manager/src/App.tsx`
- CodexPlusPlus-main：用户脚本/推荐内容入口可选复用

## 输入契约

- feature flag：`new_user_tutorial`。
- 后台可选教程文案或公告。

## 输出行为

提供：

- 修 bug、加功能、解释代码三个任务模板。
- 选择项目目录的说明。
- 如何看 Codex 计划、改动和测试结果。
- 不粘贴私密 Key、密码、支付信息的安全提示。

## 解耦要求

- 教程文案优先支持后台下发。
- 客户端内置文案只作为 fallback。
- 客户端不得硬编码套餐、价格、模型倍率、额度阈值、限流阈值或续费文案。
- 任务模板不得承诺固定模型、固定额度、固定价格或固定响应能力；这些信息必须由后台配置和服务状态决定。

## 禁止改动范围

- 不改 Codex 注入脚本。
- 不自动修改用户项目。
- 不实现在线客服。

## 测试要求

- 首次显示、关闭后记忆、后台关闭入口。
- 模板复制不包含敏感信息。
- 移动/桌面布局不溢出。

## 交付物

- 新手教程视图。
- 模板数据结构。
- 手动验收截图说明。
