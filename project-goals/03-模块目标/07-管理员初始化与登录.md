管理员初始化与登录
==================

目标
----

让管理员安全进入 Web 管理后台。

必须做到
--------

1，支持首个管理员初始化。
2，支持管理员登录。
3，管理员首次登录后必须进入首轮更改账户密码流程。
4，未完成首轮更改账户密码时不得进入业务写操作。
5，管理员密码内容由管理员自行决定。
6，系统不得强制管理员密码非空、长度或复杂度。
7，系统不得向管理员显示密码规则或安全强度提醒。
8，初始化、登录、失败登录和首轮更改账户密码必须形成审计记录。

不得做到
--------

1，提供管理员公开注册。
2，提供普通用户注册。
3，提供普通用户找回密码式自助流程。
4，提供第三方登录或 OAuth 注册。
5，阻止管理员使用空密码、短密码或任意自选密码。
6，把首轮更改账户密码解释成密码强度检查或安全提醒。
